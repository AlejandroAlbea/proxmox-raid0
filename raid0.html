Configuración de RAID 0 en Proxmox

Esta guía describe los pasos para configurar dos discos en un arreglo RAID 0 en Proxmox. Los discos involucrados son:

  • sda de 1.8TB
  • sdb de 1.8TB

1. Instalar mdadm

mdadm es la utilidad de administración de RAID en Linux. Si no está instalado, puedes instalarlo con el siguiente comando:

apt update
apt install mdadm

2. Crear el RAID 0

El siguiente paso es crear el RAID 0 usando los discos sda y sdb. Ejecuta el siguiente comando:

mdadm --create --verbose /dev/md0 --level=0 --raid-devices=2 /dev/sda /dev/sdb

Explicación del comando:

  • /dev/md0: Nombre del dispositivo RAID que estamos creando.
  • --level=0: Define el nivel de RAID (0 en este caso para un arreglo RAID 0).
  • --raid-devices=2: Número de discos que formarán parte del RAID.
  • /dev/sda /dev/sdb: Los discos que serán parte del RAID.

3. Verificar el estado del RAID

Para verificar el estado de tu nuevo RAID 0, usa el siguiente comando:

cat /proc/mdstat

Esto mostrará información sobre el RAID y si está sincronizando correctamente.

4. Crear el sistema de archivos

Ahora que el RAID 0 está configurado, debes formatearlo con un sistema de archivos, como ext4:

mkfs.ext4 /dev/md0

Esto formateará el dispositivo RAID /dev/md0 con ext4.

5. Crear un punto de montaje

Crea un directorio donde se montará el RAID 0:

mkdir /mnt/raid0

6. Montar el RAID

Ahora monta el dispositivo RAID en el punto de montaje:

mount /dev/md0 /mnt/raid0

7. Montaje automático en el arranque

Para que el RAID se monte automáticamente después de reiniciar el servidor, agrega una entrada en el archivo /etc/fstab. Primero, obtén el UUID del RAID usando:

blkid /dev/md0

Luego, edita el archivo /etc/fstab:

nano /etc/fstab

Agrega la siguiente línea al final del archivo, reemplazando tu-uuid con el UUID que obtuviste del comando anterior:

UUID=tu-uuid /mnt/raid0 ext4 defaults 0 0

Guarda los cambios y cierra el archivo.

8. Verificar el montaje

Para asegurarte de que el RAID esté montado correctamente, ejecuta:

mount -a

Si no hay errores, el RAID está montado correctamente.

9. Confirmar el montaje tras reinicio

Reinicia el servidor para confirmar que el RAID 0 se monta automáticamente al arrancar:

reboot
